# Stack-Overflow-Developer-Survey

PL: Analiza wyników ankiety przeprowadzonej wśród programistów przez serwis Stack Overflow przy użyciu biblioteki Pandas.
– Praca z dużymi zbiorami danych przy użyciu biblioteki Pandas.
– Stosowałem metody statystyki opisowej i agregacji danych.
– Analizowałem miary tendencji centralnej i rozkłady procentowe.
– Grupowałem i filtrowałem dane według różnych kryteriów.
– Pracowałem z brakującymi wartościami i zbiorami.
– Wizualizowałem wyniki analizy w wygodnym formacie tabelarycznym.

Narzędzie: Pandas (metody pracy z DataFrame; metody pracy z brakującymi wartościami, zbiorami; statystyka opisowa, metody tendencji centralnej; filtrowanie danych, zliczanie wartości; praca z danymi tekstowymi, obliczanie wartości procentowych; filtrowanie danych, praca ze zmiennymi kategorycznymi; grupowanie danych, agregacja, filtrowanie; sortowanie danych, wybór najlepszych rekordów; percentyle, filtrowanie wielokrotne, obliczanie częstotliwości).


Eng: Analysis of the results of a Stack Overflow developer survey using the Pandas library.
– Worked with large datasets using Pandas.
– Applied methods of descriptive statistics and data aggregation.
– Analyzed measures of central tendency and percentage distributions.
– Performed data grouping and filtering based on various criteria.
– Worked with missing values and sets.
– Visualized the analysis results in a convenient tabular format.

Tool: Pandas (methods for working with DataFrames; methods for handling missing values and sets; descriptive statistics and measures of central tendency; data filtering and counting; working with text data and calculating percentages; data filtering and working with categorical variables; data grouping, aggregation, and filtering; data sorting, selecting top records; percentiles, multiple filtering, frequency counting).

– Робота з великими датасетами за допомогою Pandas.
– Застосовував методи описової статистики та агрегації даних.
– Аналізував міри центральної тенденції та відсоткові розподіли.
– Виконував групування та фільтрацію даних за різними критеріями.
– Працював з пропущеними значеннями та множинами.
– Візуалізував результати аналізу у зручному табличному форматі.

Інструмент: Pandas (методи для роботи з DataFrame; методи для роботи з пропущеними значеннями, множинами; описова статистика, методи центральної тенденції; фільтрація даних, підрахунок значень; робота з текстовими даними, підрахунок відсотків; фільтрація даних, робота з категоріальними змінними; групування даних, агрегація, фільтрація; сортування даних, вибірка топ-записів; перцентилі, множинна фільтрація, підрахунок частоти).
